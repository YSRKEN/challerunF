# challerunF

## 概要

- 「ペーパーチャレラン(計算迷路チャレラン)」を解くためのソルバーです
- 使い方については **usage.txt** と **format.txt** をお読みください
- 「F」は「6番目(Ver.6)」というだけの意味です
- Visual Studio 2017で開発しています
- [Jakob Progsch氏のThreadPoolライブラリ](https://github.com/progschj/ThreadPool)を使用しています
